
#구조

## 좌측메뉴에서 사용되는 폴더
```
  chart
  popup
  library
  util
  style
  etc
```


## ASSETS
  
```
  프로젝트에 필요한 fonts, img, scss넣는곳
```

## CHART

```
  차트 컴포넌트
```

## POPUP
```
  팝업 컴포넌트(모달)
```

## ETC
```
  미분류된 컴포넌트
```

## CONFIG
```
  프로젝트 세팅 관련
```
## ERROR
```
  라우터 404 에러 표시 페이지 
```
## LIBRARY
```
  다른 라이브러리 커스텀 
```
## MAIN
```
  첫 페이지
```

## STYLE
```
  스타일 컴포넌트
```

## UTIL
```
  유틸로 사용할만한 함수 (엑셀 read, write등...)
```
